# ClipStash UI Reference

来源：用户在当前重构对话中提供的 UI 截图参考。

当前工具侧没有暴露该截图的本地附件路径，因此本目录先保存可追踪的文字参考，后续拿到原始图片文件后应补充为：

- `clipstash-ui-reference.png`

视觉方向：

- 窄窗口桌面工具，目标宽度约 420px。
- 顶部白色工具栏，左侧品牌区域，右侧放总计、置顶、设置和新建入口。
- 主色使用清晰蓝色，按钮和当前 tab 用蓝色强调。
- 消息/已归档使用胶囊式分段控制。
- 主区域以紧凑消息卡片列表为核心，减少大面积说明性内容。
- 图片缩略图按三列网格展示，列表区域可独立滚动。
- 新建表单默认收起，通过“新建”按钮展开。
- 旧数据路径等开发信息降级到可折叠区域。
